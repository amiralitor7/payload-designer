Some Notes About Formatting Conventions
=======================================

Figures
-------
* Figures should be preferably in .png format
* Figures should be named in kebab-case (see https://chaseadams.io/posts/most-common-programming-case-types/)



Equation Definitions
--------------------
* Variables should be defined in the order they appear in the equation from left to right, top to bottom.
* 
* See template:

<description> \cite{eq:<equation-name>}

\begin{equation} \label{eq:<equation-name>}
    <equation>
\end{equation}

Where:
\begin{itemize}[label={}]
    \item $<var-A>$ [\si{<unit>}]: <var A description>.
    \item $<var-B>$ [\si{<unit>}]: <var B description>.
    \item $<var-C>$ [\si{<unit>}]: <var C description>.
\end{itemize}
